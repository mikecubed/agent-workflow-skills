# sdd-workflow

Companion plugin bundle for spec-driven development inside the `agent-orchestration` marketplace.

This plugin provides:

- `sdd.specify`
- `sdd.plan`
- `sdd.tasks`
- `sdd-feature-workflow`

Install locally:

```bash
copilot plugin install ./plugins/sdd-workflow
claude --plugin-dir ./plugins/sdd-workflow
```

This bundle is intentionally limited to the runtime plugin assets needed for marketplace use.
